# HelloClaude

A small Python learning project with standalone utility scripts.

## Requirements

- Python 3.9+

## Setup

```bash
python -m venv .venv
source .venv/bin/activate
```

## Scripts

### main.py
PyCharm-generated entry point with a `print_hi` function.
```bash
python main.py
```

### sum.py
Calculates the sum of a list of numbers.
```bash
python sum.py
```

### factorial.py
Computes the factorial of a number recursively. Prompts for user input at runtime.
```bash
python factorial.py
```
